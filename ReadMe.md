# CRUD GraphQL API

## Note: Please make the .env file before using. Follow the .env.txt file.

To use:

    * npm install
        - to install dependencies
    * npm run build
        - to generate compiled project
    * npm start
        - to run the project
    * npm run dev
        - to be responsive during development
